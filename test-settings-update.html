📋 New Features in General Tab

  • Weekend Days Selection: Managers can select multiple days as weekends
  • Holiday Calendar: Interactive calendar to select specific dates as holidays
  • Selected Holidays Display: Visual tags showing selected holiday dates
  • Save Settings: Persist weekend and holiday settings to server
  • Load Settings: Automatically load saved settings when tab is opened

🔗 Test the Features

To test the new features:

  1. Open the application at http://localhost:3000
  2. Sign in as a Manager
  3. Navigate to Settings (الإعدادات)
  4. You'll see the updated tabs:
    • Personal Info (المعلومات الشخصية) - formerly General
    • General (عام) - new system settings
    • Security (الأمان)
    • Notifications (الإشعارات)
  5. Click on the new General tab to test:
    • Select weekend days by clicking checkboxes
    • Navigate through months using arrow buttons
    • Click on calendar dates to select holidays
    • Remove holidays by clicking the × on holiday tags
    • Save settings and verify they persist
